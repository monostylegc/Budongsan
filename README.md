# 한국 부동산 시장 ABM 시뮬레이션

Agent-Based Model을 이용한 한국 부동산 시장 시뮬레이션입니다.

## 설치

```bash
uv sync
```

## 실행

```bash
# 기본 시나리오 (10년)
uv run python main.py --scenario baseline

# 시나리오 비교 (다주택자 규제 효과)
uv run python main.py --scenario comparison

# 금리 충격 시나리오
uv run python main.py --scenario interest

# 전체 시나리오
uv run python main.py --scenario all
```

## 모델 개요

### 에이전트
- **가구 (Households)**: 100만 가구, 연령/소득/자산/주택보유 속성
- **주택 (Houses)**: 50만 채, 지역/가격/면적 속성

### 지역 (13개)
| ID | 지역 | 기준가격 |
|----|------|----------|
| 0 | 강남3구 | 20억 |
| 1 | 마용성 | 15억 |
| 2 | 기타서울 | 8억 |
| 3 | 분당판교 | 12억 |
| 4-6 | 경기/인천 | 4-6억 |
| 7-12 | 지방 | 2-4.5억 |

### 정책 변수
- **대출 규제**: LTV, DTI
- **세금**: 취득세, 양도세, 종부세
- **금리**: 기준금리, 주담대 스프레드

```bash

# 가설 3: 금리 vs 규제

# 가설 4: 종부세 효과
uv run python main.py --scenario h4

```

## 검증 가설 및 결과

### 가설 1: 다주택자 규제의 역설 (양도세 40%/60%/80%)
| 시나리오 | 강남 변화율 | 거래량 | 다주택자 비율 |
|---------|-----------|--------|--------------|
| 양도세 40% | +4.2% | 17,619 | 14.0% |
| 양도세 60% | +4.2% | 15,755 | 14.6% |
| 양도세 80% | +4.2% | 13,903 | 14.8% |

**결론**: 양도세 중과는 거래량 감소(매물 잠김)를 초래하지만 가격 효과는 미미

### 가설 2: 풍선효과의 비선형성 (LTV 0%/20%/40%)
| 시나리오 | 서울 | 수도권 | 지방 | 풍선효과 지수 |
|---------|-----|-------|------|-------------|
| LTV 0% | -23.2% | -7.3% | -2.9% | +20.4%p |
| LTV 20% | +2.0% | +6.6% | +7.9% | +5.9%p |
| LTV 40% | +8.7% | +13.1% | +12.3% | +3.6%p |

**결론**: 과도한 규제(LTV 0%)는 풍선효과를 기하급수적으로 증폭

### 가설 3: 금리 vs 규제
| 시나리오 | 전국 평균 | 거래량 | 비고 |
|---------|----------|--------|------|
| 금리 3% + 강규제 | +11.6% | 14,500 | 최악 |
| 금리 5% + 중규제 | +10.8% | 17,611 | - |
| 금리 7% + 약규제 | +10.2% | 18,852 | 최적 |
| 금리 7% + 강규제 | +10.0% | 13,595 | - |

**결론**: 금리 인상이 규제보다 효과적이며 부작용이 적음

### 가설 4: 종부세 효과 (1%/3%/6%)
| 시나리오 | 거래량 | 다주택자 감소 |
|---------|--------|--------------|
| 종부세 1% | 14,136 | -0.2%p |
| 종부세 3% | 16,747 | -0.6%p |
| 종부세 6% | 18,532 | -0.8%p |

**결론**: 종부세 강화는 거래 활성화에 기여하나 단독으로는 효과 제한적

자세한 분석은 [docs/experiment_report.md](docs/experiment_report.md) 참조

## 학술적 근거

이 모델은 다음 학술 연구에 기반합니다:

### 손실 회피 (Loss Aversion)
- **Tversky & Kahneman (1992)**: 전망이론, lambda = 2.25
- **Genesove & Mayer (2001, QJE)**: 부동산 시장 실증, lambda = 2.5
  - 손실 상황 매도자: 호가 25-35% 프리미엄
  - 매도 확률 급감 (손해보고 못 팔음)

### ABM 참고 문헌
- **JASSS 2020**: 한국 주택시장 LTV/DTI ABM
- **JASSS 2024**: UK 금융충격 ABM
