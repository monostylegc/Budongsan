## 3. 수정된 파라미터

### 3.1 지역별 주택 가격 (`config.py:REGIONS`)

```python
# Before → After (만원 단위)

마용성:  150000 → 200000  # 15억 → 20억
기타서울: 80000 → 120000  # 8억 → 12억
분당판교: 120000 → 150000 # 12억 → 15억
경기남부: 60000 → 70000   # 6억 → 7억
경기북부: 40000 → 45000   # 4억 → 4.5억
인천:    50000 → 50000    # 5억 (유지)
부산:    45000 → 50000    # 4.5억 → 5억
대구:    35000 → 47000    # 3.5억 → 4.7억
광주:    30000 → 38000    # 3억 → 3.8억
대전:    32000 → 44000    # 3.2억 → 4.4억
세종:    45000 → 50000    # 4.5억 → 5억

```

### 3.2 소득/자산 분포 (`config.py:AgentCompositionConfig`)

```python
# 소득 분포

# 자산 분포

# 주택소유 분포

initial_one_house_rate: 0.40 → 0.42
```

## 6. 시뮬레이션 결과 (24개월, 새 파라미터)

### 지역별 가격 변화

| 지역 | 초기 가격 | 최종 가격 | 변화율 |
|------|----------|----------|--------|
| 강남3구 | 30.0억 | 30.9억 | **+3.16%** |
| 마용성 | 20.0억 | 19.7억 | -1.46% |
| 기타서울 | 12.0억 | 12.1억 | +1.20% |
| 분당판교 | 15.0억 | 18.2억 | **+21.43%** |
| 경기남부 | 7.0억 | 7.8억 | +11.75% |
| 경기북부 | 4.5억 | 5.9억 | +32.14% |
| 인천 | 5.0억 | 7.0억 | +40.03% |
| 부산 | 5.0억 | 6.6억 | +32.94% |
| 대구 | 4.7억 | 5.5억 | +17.15% |
| 광주 | 3.8억 | 5.0억 | +31.65% |
| 대전 | 4.4억 | 5.5억 | +26.07% |
| 세종 | 5.0억 | 7.1억 | +42.49% |
| 기타지방 | 2.5억 | 3.3억 | +33.36% |
| **전국 평균** | **9.1억** | **10.4억** | **+13.49%** |

### 분석

1. **강남 안정화**: 가격 상승 억제, 고가 지역 조정 (+3%)
2. **마용성 소폭 하락**: 구매력 대비 높은 가격 조정 (-1.5%)
3. **분당/판교 강세**: 테크노밸리 일자리 밀도 반영 (+21%)
4. **지방 강세**: 상대적으로 저렴 + 구매력 충분 (+17~40%)
5. **전국 평균 상승**: 인플레이션 기대 + 실수요 반영 (+13.5%)

### 이전 결과 (구 파라미터) vs 현재 결과 비교

| 지표 | 이전 (구 파라미터) | 현재 (새 파라미터) |
|------|------------------|------------------|
| 강남 변화 | -6.23% | **+3.16%** |
| 전국 평균 | +0.49% | **+13.49%** |
| 시장 활성도 | 낮음 | 높음 |

**결론**: 실제 한국 데이터 기반 캘리브레이션으로 시뮬레이션이 더 현실적으로 작동
